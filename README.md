# SuperApp - Модульное десктопное приложение на Python

SuperApp - это модульное десктопное приложение, разработанное на Python с использованием фреймворка PySide6. Приложение представляет собой платформу для полезных утилит, включающую трекер валют с интерактивными графиками и менеджер бюджета с системой учёта доходов и расходов.

## Описание проекта



## Описание утилит

### Утилита 1: Трекер валют

Модуль для отслеживания динамики курсов валют. Позволяет пользователю:
- Выбирать валюту из списка (USD, EUR, CNY, GBP, JPY)
- Задавать период наблюдения (от 7 до 365 дней)
- Получать актуальные данные от Центрального Банка РФ через API
- Видеть интерактивный график изменения курса
- Анализировать статистику: текущий курс, изменение за день в процентах, минимум и максимум за период
- Конвертировать сумму из выбранной валюты в рубли по актуальному курсу
- Визуально оценивать тренд через цветовую индикацию (зеленый - рост, красный - падение)

### Утилита 2: Бюджет и накопления

Модуль для управления личными финансами и планирования крупных покупок. Позволяет пользователю:
- Добавлять доходы с категориями (Стипендия, Подработка, Подарки, Другое)
- Добавлять расходы с категориями (Еда, Транспорт, Развлечения, Одежда, Учеба, Другое)
- Видеть текущий баланс в реальном времени
- Просматривать список последних 10 операций с цветовой индикацией доходов и расходов
- Анализировать структуру расходов через круговую диаграмму по категориям за текущий месяц
- Создавать цели накоплений с указанием целевой суммы
- Отслеживать прогресс достижения целей через прогресс-бары
- Обновлять сумму накоплений по каждой цели
- Сохранять все данные локально в JSON-файле для использования между запусками приложения

### Заглушки для будущих утилит

Приложение содержит 3 заглушки для будущих модулей, демонстрирующие готовность архитектуры к расширению.

## Обоснование выбора технологий

### UI-фреймворк: PySide6

Для разработки графического интерфейса выбран фреймворк PySide6 (официальная библиотека Qt для Python) по следующим причинам:

1. **Модульность и масштабируемость**: Архитектура сигналов и слотов (Signals & Slots) в Qt идеально подходит для концепции SuperApp, позволяя модулям общаться с ядром без жестких зависимостей.

2. **Производительность**: Qt является одним из самых производительных GUI-фреймворков, что критично для отрисовки интерактивных графиков в реальном времени.

3. **Кроссплатформенность**: Приложение работает на Windows, macOS и Linux без изменений в коде.

4. **Лицензия LGPL**: PySide6 распространяется под лицензией LGPL, что позволяет использовать его как в открытых, так и в коммерческих проектах.

5. **Богатый набор компонентов**: Встроенные виджеты для таблиц, деревьев, графиков и других элементов интерфейса.

### Библиотека для графиков: pyqtgraph и matplotlib

Для отрисовки графиков используются две библиотеки:

**pyqtgraph** (для трекера валют):
- Высокая производительность с использованием NumPy
- Отрисовка с частотой 60 FPS
- Нативная интеграция с PySide6

**matplotlib** (для менеджера бюджета):
- Возможность создания круговых диаграмм
- Богатые возможности кастомизации
- Стабильность и широкое сообщество

### Дополнительные библиотеки

- **requests**: Для HTTP-запросов к API Центрального Банка РФ
- **pandas**: Для обработки и анализа исторических данных

## Архитектура приложения

Приложение построено по паттерну MVC (Model-View-Controller) с элементами модульной архитектуры:

### Ядро (Core)

Ядро приложения отвечает за:
- Инициализацию QApplication и глобальных настроек
- Управление навигацией между модулями
- Единый интерфейс для работы с внешними API

### Модули (Modules)

Каждый модуль-утилита состоит из:
- **View**: Отвечает за отображение интерфейса и обработку пользовательского ввода
- **Controller**: Содержит бизнес-логику и координирует работу между View и данными
- **Model** (для менеджера бюджета): Отвечает за хранение и обработку данных

Модули изолированы друг от друга и общаются с ядром через сигналы и слоты.

### Компоненты (Components)

Переиспользуемые UI-компоненты, которые могут использоваться в разных модулях (например, виджет для графиков).

## Установка и запуск

### Системные требования

- Python 3.8 или выше
- Windows, macOS или Linux

### Установка

1. Клонируйте репозиторий:
```
git clone <url-репозитория>
cd SuperApp
```

2. Создайте виртуальное окружение:
```
python -m venv venv
```

3. Активируйте виртуальное окружение:

Windows:
```
venv\Scripts\activate
```

macOS/Linux:
```
source venv/bin/activate
```

4. Установите зависимости:
```
pip install -r requirements.txt
```

## Использование

### Навигация

Приложение имеет боковую панель навигации слева. Для переключения между утилитами кликните на нужный пункт меню.

### Трекер валют

1. Выберите валюту из выпадающего списка (USD, EUR, CNY, GBP, JPY)
2. Укажите количество дней для отображения истории (от 7 до 365)
3. Нажмите "Обновить график" для загрузки данных
4. Используйте калькулятор конвертации для перевода суммы в рубли

### Бюджет и накопления

**Вкладка "Операции":**
1. Выберите тип операции (Доход или Расход)
2. Выберите категорию из списка
3. Введите сумму и описание
4. Нажмите "Добавить"
5. Наблюдайте за обновлением баланса, таблицы операций и круговой диаграммы

**Вкладка "Цели":**
1. Введите название цели (например, "Ноутбук")
2. Укажите целевую сумму
3. Нажмите "Добавить цель"
4. Обновляйте сумму накоплений через поле "Накоплено" и кнопку "Сохранить"

### Интерактивный график

- Наведите курсор на точку графика для просмотра точного значения
- Используйте колесико мыши для масштабирования
- Перетаскивайте график для перемещения по временной шкале

## Тестирование

Проект покрыт юнит-тестами для проверки корректности работы ключевых компонентов:

### Покрытие тестами

- **core/api_client.py**: Тесты получения данных от ЦБ РФ (5 тестов)
  - Успешное получение текущих курсов
  - Обработка ошибок сети
  - Получение курса конкретной валюты

  - Получение исторических данных

  - Успешная загрузка данных
  - Обработка пустой истории
  - Успешная конвертация валюты
  - Обработка ошибок при конвертации

- **modules/budget_manager/models.py**: Тесты модели бюджета (8 тестов)
  - Добавление доходов и расходов
  - Расчет положительного и отрицательного баланса
  - Получение последних транзакций
  - Добавление и обновление целей
  - Получение расходов за текущий месяц

- **modules/budget_manager/controller.py**: Тесты контроллера бюджета (4 теста)
  - Добавление доходов и расходов через контроллер
  - Добавление и обновление целей
  - Обновление интерфейса

Всего: 21 юнит-тест





## API и источники данных

Приложение использует бесплатный API Центрального Банка РФ через сервис cbr-xml-daily.ru:

- **Текущие курсы**: https://www.cbr-xml-daily.ru/daily_json.js
- **Исторические данные**: https://www.cbr-xml-daily.ru/archive/{год}/{месяц}/{день}/daily_json.js

API предоставляет данные без необходимости регистрации и API-ключей.

Данные менеджера бюджета сохраняются локально в файле budget_data.json в формате JSON.
